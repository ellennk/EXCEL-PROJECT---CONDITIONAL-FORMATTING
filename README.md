# EXCEL-PROJECT---CONDITIONAL-FORMATTING
 This project covers a a powerful spreadsheet tool (found in Excel, Google Sheets, etc.) that automatically changes the appearance of cells—such as background color, font style, or border—based on the data they contain. 

   It is used to quickly highlight, emphasize, or identify patterns, trends, and exceptions in datasets, making large amounts of information easier to analyze at a glance


### KEY QUESTIIONS ON CONDITIONAL FORMATTING
1. Apply bold text + background color to column headers.

2.  Format Cost as currency with 2 decimal places.
   
3. Format VisitDate as dd-mmm-yyyy.

4. Highlight patients older than 60; Highlight treatments
costing more than 3000.

5. Hightlight all cells in diagnosis column with flu.

6. Use Excel functions Vlook up,Xlookup, index match and Hlookup to identify specific cells.

###  GENERAL SIGNIFICANCE OF CONDITIONAL FORMATTING
1.Efficient Data Analysis: It saves significant time and effort by automating the formatting process. Instead of manually scanning thousands of rows, users can set rules to automatically highlight critical data points like sales targets or overdue deadlines.


2.Quick Identification of Trends and Patterns: By using visual aids like color scales, data bars, and icon sets, it makes patterns and trends (such as rising or falling sales) immediately apparent.

3.Enhanced Error and Anomaly Detection: It acts as a built-in alert system to quickly spot outliers, duplicates, or inconsistent data. For example, you can set a rule to highlight any negative values or duplicate ID numbers in red.

4.Improved Data Presentation and Engagement: Conditional formatting makes spreadsheets more visually appealing and easier for others to understand. This increased clarity improves user engagement and helps stakeholders interpret complex datasets without needing deep technical analysis
